# SQL Analysis

## Objective
Perform SQL analysis on an E-Commerce dataset using PostgreSQL.

## Tools Used
- PostgreSQL
- pgAdmin 4

## Queries Performed
1. Total Orders
2. Total Revenue
3. Average Order Value
4. Top Selling Products
5. Highest Revenue Products
6. Payment Method Analysis
7. Order Status Analysis
8. Referral Source Analysis
9. Coupon Usage Analysis
10. Monthly Revenue Trend

## Files Included
- SQL_Queries.sql
- SQL_Analysis_Report.docx

## Outcome
Generated business insights using SQL queries on the E-Commerce dataset.
